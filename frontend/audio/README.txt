Drop your Pokemon OST .mp3 files into the matching generation subfolder:
Generation 1/, Generation 2/, Generation 3/ ... Generation 9/

Each file's name must exactly match the "filename" column (minus the
"Generation N/" prefix) in backend/data/catalog.txt or backend-mysql/schema.sql,
e.g. Generation 1/Pallet Town.mp3 for a track listed as Generation 1/Pallet Town.mp3.
